# Play Store Listing - VoidBoard

## Short Description

Play VoidBoard chess with offline matches, AI, optional Google sign-in, undo, and saved games.

## Full Description

VoidBoard is a clean, privacy-friendly chess game made for Android players who want classic chess with optional Google sign-in and offline guest play.

Play against a friend on the same device or challenge the built-in AI across Easy, Medium, and Hard levels. The app validates legal moves, highlights available moves, tracks move history, supports undo, and saves your current game locally so you can resume later.

Designed for quick casual games and focused practice, VoidBoard includes a polished void-black neon board UI, pawn promotion, castling, en passant, check, checkmate, stalemate, and draw detection.

No account is required for local play. Internet is used only when you choose to sign in with Google.

## Privacy Policy Draft

VoidBoard is a chess game with offline guest play and optional Google sign-in. The app does not require users to create an account for local play. The app requests internet only for optional Google sign-in and does not request location, contacts, camera, microphone, or storage permissions.

The app stores user settings, saved chess game state, and optional Google profile display data locally on the user's device using Android DataStore. This local data may include board theme, piece style, sound preference, AI difficulty, current board position, move history, Google display name, email address, and profile image URL.

VoidBoard does not sell personal data. The current release does not include advertising SDKs, analytics SDKs, cloud game sync, or payment processing. Google sign-in is handled by Google services and is optional.

If future versions add optional ads, analytics, cloud sync, or premium purchases, this privacy policy should be updated before release to describe any new data practices.

## Permissions Note

The source manifest requests internet for optional Google sign-in. It does not request location, contacts, camera, microphone, media, or storage permissions. AndroidX may add an app-private signature permission for dynamic receiver safety in the merged manifest; this is not a user-facing dangerous permission and does not grant access to personal data.
